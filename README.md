# Inventory Management System

A Java-based Inventory Management System developed using JDBC and MySQL.  
This project allows users to manage products, suppliers, stock quantity, price details, and low-stock alerts.

## Features
- Add new products
- View all products
- Search product by ID
- Update product quantity and price
- Delete product records
- Low stock alert
- JDBC connectivity with MySQL

## Technologies Used
- Java
- JDBC
- MySQL
- OOP Concepts
- DAO Design Pattern
